⇩ Cargar Datos desde BigQuery

Ejecuta cada query en BigQuery, descarga como JSON y reemplaza los arrays en la sección DATA del script. Importante: cada registro ahora debe incluir el campo mes (formato YYYY-MM) para que el filtro de período funcione.

1
Abre fbm_bigquery_queries.sql — las queries ya incluyen el campo mes.
2
Ejecuta Q1-Q8 y descarga cada resultado como JSON Newline delimited.
3
Abre el HTML en VS Code, busca // ===== DATA ===== y reemplaza cada array.
4
Cambia IS_DEMO: true a IS_DEMO: false.
